# Events Catalog

 Event catalog. For example i want to see all crypto events in the prague in next month with the links to the events and some event name and description and location on the map. Allow people to submit the events by them selves with dashboard overview and event manager, and admin page to manage the data or other users. The domains specific events like crypto or ai or cooking will be set by the administrator and will be in specific subdomain.

## Project Structure

- `projects/events-frontend/` - Vue.js frontend application
- `projects/EventsApi/` - C# BE
